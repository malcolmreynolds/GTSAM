gtsam::GenericSequentialSolver< FACTOR > Class Template Reference

Detailed Description

template<class FACTOR>
class gtsam::GenericSequentialSolver< FACTOR >

This solver implements sequential variable elimination for factor graphs.

Underlying this is a column elimination tree, see Gilbert 2001 BIT.

The elimination ordering is "baked in" to the variable indices at this stage, i.e. elimination proceeds in order from '0'.

This is not the most efficient algorithm we provide, most efficient is the MultifrontalSolver, which examines and uses the clique structure. However, sequential variable elimination is easier to understand so this is a good starting point to learn about these algorithms and our implementation. Additionally, the first step of MFQR is symbolic sequential elimination.
