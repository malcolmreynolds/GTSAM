gtsam::IndexConditional Class Reference

Detailed Description

IndexConditional serves two purposes.

It is the base class for GaussianConditional, and also functions as a symbolic conditional with Index keys, produced by symbolic elimination of IndexFactor.

It derives from Conditional with a key type of Index, which is an unsigned integer.
