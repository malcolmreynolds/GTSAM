template<class CONDITIONAL , class CLIQUE >
void gtsam::BayesTree< CONDITIONAL, CLIQUE >::insert ( const sharedClique &  subtree)

Hang a new subtree off of the existing tree.

This finds the appropriate parent clique for the subtree (which may be the root), and updates the nodes index with the new cliques in the subtree. None of the frontal variables in the subtree may appear in the separators of the existing BayesTree.
