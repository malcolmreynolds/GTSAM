std::pair< SymbolicFactorGraph::sharedConditional, SymbolicFactorGraph > gtsam::SymbolicFactorGraph::eliminate ( const std::vector< Index > &  variables) const

Factor the factor graph into a conditional and a remaining factor graph.

Given the factor graph \( f(X) \), and variables to factorize out \( V \), this function factorizes into \( f(X) = f(V;Y)f(Y) \), where \( Y := X\V \) are the remaining variables. If \( f(X) = p(X) \) is a probability density or likelihood, the factorization produces a conditional probability density and a marginal \( p(X) = p(V|Y)p(Y) \).

For efficiency, this function treats the variables to eliminate variables as fully-connected, so produces a dense (fully-connected) conditional on all of the variables in variables, instead of a sparse BayesNet. If the variables are not fully-connected, it is more efficient to sequentially factorize multiple times. Note that this version simply calls FactorGraph<GaussianFactor>::eliminate with EliminateSymbolic as the eliminate function argument.

std::pair< SymbolicFactorGraph::sharedConditional, SymbolicFactorGraph > gtsam::SymbolicFactorGraph::eliminateFrontals ( size_t  nFrontals) const

Eliminate the first n frontal variables, returning the resulting conditional and remaining factor graph - this is very inefficient for eliminating all variables, to do that use EliminationTree or JunctionTree.

Note that this version simply calls FactorGraph<IndexFactor>::eliminateFrontals with EliminateSymbolic as the eliminate function argument.
