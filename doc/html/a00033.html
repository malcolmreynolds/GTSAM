gtsam::Conditional< KEY > Class Template Reference

Detailed Description

template<typename KEY>
class gtsam::Conditional< KEY >

Base class for conditional densities, templated on KEY type.

This class provides storage for the keys involved in a conditional, and iterators and access to the frontal and separator keys.

Derived classes must redefine the Factor and shared_ptr typedefs to refer to the associated factor type and shared_ptr type of the derived class. See IndexConditional and GaussianConditional for examples.
