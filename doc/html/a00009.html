Detailed Description

template<class CONDITIONAL>
class gtsam::BayesNet< CONDITIONAL >

A BayesNet is a list of conditionals, stored in elimination order, i.e.

leaves first, parents last. GaussianBayesNet and SymbolicBayesNet are defined as typedefs of this class, using GaussianConditional and IndexConditional as the CONDITIONAL template argument.

template<class CONDITIONAL >
void gtsam::BayesNet< CONDITIONAL >::popLeaf ( iterator  conditional)

Remove any leaf conditional.

The conditional to remove is specified by iterator. To find the iterator pointing to the conditional containing a particular key, use find(), which has \( O(n) \) complexity. The popLeaf function by itself has \( O(1) \) complexity.

If gtsam is compiled with GTSAM_EXTRA_CONSISTENCY_CHECKS defined, this function will check that the node is indeed a leaf, but otherwise will not check, because the check has \( O(n^2) \) complexity.

Example 1:

// Remove a leaf node with a known conditional
GaussianBayesNet gbn = ...
GaussianBayesNet::iterator leafConditional = ...
gbn.popLeaf(leafConditional);

Example 2:

// Remove the leaf node containing variable index 14
GaussianBayesNet gbn = ...
gbn.popLeaf(gbn.find(14));
