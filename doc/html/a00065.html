Detailed Description

template<class FACTOR>
class gtsam::FactorGraph< FACTOR >

A factor graph is a bipartite graph with factor nodes connected to variable nodes.

In this class, however, only factor nodes are kept around.

Factor the factor graph into a conditional and a remaining factor graph.

Given the factor graph \( f(X) \), and variables to factorize out \( V \), this function factorizes into \( f(X) = f(V;Y)f(Y) \), where \( Y := X\V \) are the remaining variables. If \( f(X) = p(X) \) is a probability density or likelihood, the factorization produces a conditional probability density and a marginal \( p(X) = p(V|Y)p(Y) \).

For efficiency, this function treats the variables to eliminate variables as fully-connected, so produces a dense (fully-connected) conditional on all of the variables in variables, instead of a sparse BayesNet. If the variables are not fully-connected, it is more efficient to sequentially factorize multiple times.
